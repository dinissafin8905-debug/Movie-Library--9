## 📌 О проекте

**Movie Library** - это desktop-приложение для хранения информации о фильмах. Программа помогает вести личную коллекцию фильмов, отслеживать рейтинги, анализировать предпочтения по жанрам и годам выпуска.

### 🎯 Основные возможности

| Функция | Описание |
|---------|----------|
| ➕ **Добавление фильмов** | Запись информации о фильме (название, жанр, год, рейтинг) |
| 🔍 **Фильтрация** | По жанру и по году выпуска |
| 📊 **Статистика** | Средний рейтинг, лучший/худший фильм, количество по жанрам |
| 💾 **JSON** | Сохранение и загрузка данных |
| 📋 **Таблица** | Удобный просмотр всех фильмов |
| 🗑️ **Очистка** | Удаление всех записей |

### 📂 Доступные жанры

| Жанр | Эмодзи | Описание |
|------|--------|----------|
| Боевик | 🔫 | Экшн, перестрелки, погони |

| Триллер | 🔪 | Напряжённые, захватывающие |

| Приключения | 🗺️ | Путешествия, поиски |
| Детектив | 🕵️ | Расследования, загадки |
| Анимация | 🐭 | Мультфильмы |

## 🚀 Установка и запуск

### Системные требования
- Python 3.8 или выше
- Tkinter (входит в стандартную поставку Python)

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/your-username/movie-library.git

# Переход в папку проекта
cd movie-library

# Запуск приложения
python movie_library.py
Никаких дополнительных библиотек не требуется!

📖 Как использовать
1️⃣ Добавление фильма
Шаг	Действие	Пример
1	Введите название	Побег из Шоушенка
2	Выберите жанр	Драма
3	Введите год	1994
4	Введите рейтинг	9.3
5	Нажмите "ДОБАВИТЬ ФИЛЬМ"	✅
Правила ввода:

Название: любое (не может быть пустым)

Жанр: выбирается из списка

Год: число от 1888 до 2026

Рейтинг: число от 0 до 10 (можно с десятичной точкой, например: 8.5)

2️⃣ Фильтрация фильмов
По жанру:

Выберите жанр в списке фильтра

Нажмите "ПРИМЕНИТЬ ФИЛЬТР"

В таблице останутся только фильмы выбранного жанра

По году выпуска:

Введите "Год от" (начальный год)

Введите "Год до" (конечный год)

Нажмите "ПРИМЕНИТЬ ФИЛЬТР"

Сброс фильтра:

Нажмите "СБРОСИТЬ ФИЛЬТР"

3️⃣ Просмотр статистики
Статистика обновляется автоматически и показывает:

Всего фильмов - количество фильмов в коллекции

Средний рейтинг - средняя оценка всех фильмов

Лучший фильм - фильм с самым высоким рейтингом

Худший фильм - фильм с самым низким рейтингом

По жанрам - количество фильмов в каждом жанре

4️⃣ Работа с JSON
Кнопка	Действие
💾 СОХРАНИТЬ В JSON	Сохранить все фильмы в файл movies.json
📂 ЗАГРУЗИТЬ ИЗ JSON	Загрузить фильмы из выбранного файла
🗑️ ОЧИСТИТЬ ВСЁ	Удалить все фильмы
🧪 Примеры использования
Пример 1: Создание коллекции любимых фильмов
python
1. Добавить: "Побег из Шоушенка", Драма, 1994, 9.3
2. Добавить: "Крёстный отец", Драма, 1972, 9.2
3. Добавить: "Тёмный рыцарь", Боевик, 2008, 9.0
Результат: В таблице 3 фильма, средний рейтинг ≈ 9.2

Пример 2: Поиск фильмов по жанру
python
1. В фильтре выбрать жанр "Комедия"
2. Нажать "ПРИМЕНИТЬ ФИЛЬТР"
Результат: Показаны только комедии

Пример 3: Поиск фильмов 90-х годов
python
1. Год от: 1990
2. Год до: 1999
3. Нажать "ПРИМЕНИТЬ ФИЛЬТР"
Результат: Показаны фильмы 1990-1999 годов

Пример 4: Сохранение и восстановление данных
python
1. Добавить несколько фильмов
2. Нажать "СОХРАНИТЬ В JSON"
3. Закрыть программу
4. Открыть программу снова
5. Нажать "ЗАГРУЗИТЬ ИЗ JSON"
Результат: Все фильмы восстановлены

🧪 Тесты для проверки работы
Тест 1: Проверка ввода года
Действие	Ожидаемый результат
Ввести abc	❌ Ошибка: "Год должен быть числом!"
Ввести 1800	❌ Ошибка: "Год должен быть от 1888 до 2026!"
Ввести 2020	✅ Фильм добавлен
Тест 2: Проверка ввода рейтинга
Действие	Ожидаемый результат
Ввести abc	❌ Ошибка: "Рейтинг должен быть числом!"
Ввести -1	❌ Ошибка: "Рейтинг должен быть от 0 до 10!"
Ввести 8.5	✅ Фильм добавлен
Тест 3: Проверка фильтрации по жанру
Действие	Ожидаемый результат
Добавить: Боевик, Комедия, Драма	3 фильма в таблице
Выбрать фильтр "Боевик" → Применить	Показан только 1 фильм
Нажать "СБРОСИТЬ ФИЛЬТР"	Показаны все 3 фильма
Тест 4: Проверка фильтрации по году
Действие	Ожидаемый результат
Добавить: 1990, 1995, 2000, 2005	4 фильма
Фильтр: год от 1995 до 2000	Показаны фильмы 1995 и 2000
Тест 5: Проверка статистики
Действие	Ожидаемый результат
Добавить рейтинги: 9.0, 8.0, 7.0	
Всего фильмов: 3	✓
Средний рейтинг: 8.0	✓
Лучший фильм: 9.0	✓
Худший фильм: 7.0	✓
Тест 6: Проверка сохранения и загрузки JSON
Действие	Ожидаемый результат
Добавить 3 фильма → Сохранить	✅ Файл создан
Очистить всё → Загрузить	✅ Все 3 фильма восстановлены

❗ Возможные ошибки и их решение
Ошибка	Решение
"Год должен быть числом"	Введите только цифры, например: 1994
"Год должен быть от 1888 до 2026"	Введите год в диапазоне от 1888 до текущего года
"Рейтинг должен быть числом"	Введите число, например: 8.5 или 9
"Рейтинг должен быть от 0 до 10"	Введите число от 0 до 10
"Не удалось сохранить файл"	Проверьте права на запись в папке программы
