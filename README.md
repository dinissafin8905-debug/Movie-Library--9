## 📋 О проекте

**Movie Library** — это десктопное приложение для хранения информации о фильмах с удобным интерфейсом, разработанное на Python с использованием библиотеки Tkinter. Программа помогает вести личную коллекцию фильмов, отслеживать рейтинги и анализировать предпочтения.

**Автор:** Суроян Роман Асланович, 8 класс, 2026 г.

## ✨ Возможности

- ✅ **Добавление фильмов** — название, жанр, год выпуска, рейтинг
- ✅ **Категоризация** — 10 жанров (Боевик, Комедия, Драма, Фантастика, Ужасы, Триллер, Мелодрама, Приключения, Детектив, Анимация)
- ✅ **Фильтрация** — по жанрам и диапазону годов выпуска
- ✅ **Статистика** — средний рейтинг, лучший/худший фильм, распределение по жанрам
- ✅ **Визуализация рейтинга** — звёздочки (⭐) для наглядной оценки
- ✅ **Сохранение/загрузка** — в формате JSON
- ✅ **Автоматическое сохранение** — данные не теряются при закрытии
- ✅ **Встроенная инструкция** — с полосой прокрутки
- ✅ **Современный дизайн** — тёмная тема с цветовыми акцентами

🐛 Устранение проблем с зависимостями
❌ Ошибка: "No module named tkinter"
Причина: Tkinter не установлен

Решение:

Windows: Переустановите Python, убедитесь, что в установщике отмечена опция "tcl/tk" и "IDLE"

Ubuntu/Debian: sudo apt-get install python3-tk

Fedora: sudo dnf install python3-tkinter

Arch: sudo pacman -S tk

macOS: brew install python-tk

❌ Ошибка: "Python not found"
Причина: Python не добавлен в PATH

Решение:

Windows: Переустановите Python и отметьте "Add Python to PATH"

macOS/Linux: Используйте python3 вместо python

❌ Ошибка: "SyntaxError: invalid syntax"
Причина: Используется старая версия Python

Решение: Обновите Python до версии 3.8 или выше

❌ Ошибка при импорте модулей
Все используемые модули входят в стандартную библиотеку Python, поэтому дополнительная установка не требуется:

🎮 Как использовать
1️⃣ Добавление фильма
Введите название фильма (например, "Побег из Шоушенка")

Выберите жанр из выпадающего списка

Укажите год выпуска (от 1888 до 2026)

Укажите рейтинг (от 0 до 10, можно использовать десятичные: 8.5)

Нажмите «ДОБАВИТЬ ФИЛЬМ»

2️⃣ Фильтрация данных
По жанру:

Выберите жанр в выпадающем списке

Нажмите «ПРИМЕНИТЬ»

По году:

Укажите "Год от" и "Год до"

Нажмите «ПРИМЕНИТЬ»

Комбинированный фильтр:

Можно одновременно фильтровать по жанру и годам

Сбросить фильтры:

Нажмите «СБРОСИТЬ»

3️⃣ Анализ статистики
Статистика обновляется автоматически и показывает:

Всего фильмов - количество в коллекции (с учётом фильтра)

Средний рейтинг - средняя оценка всех фильмов

Лучший фильм - фильм с самым высоким рейтингом

Худший фильм - фильм с самым низким рейтингом

По жанрам - количество фильмов в каждом жанре

4️⃣ Визуализация рейтинга
Рейтинг отображается в таблице в виде звёздочек:

⭐ - 2 балла

☆ - 0.5 балла

Пример: рейтинг 8.5 → ⭐⭐⭐⭐☆

5️⃣ Работа с JSON
Кнопка	Действие
💾 СОХРАНИТЬ	Сохранить коллекцию в файл movies.json
📂 ЗАГРУЗИТЬ	Загрузить коллекцию из JSON файла
🗑️ ОЧИСТИТЬ	Удалить все фильмы
